The function \( f(x) \) is defined such that for any real number \( x \), \( f(x + 1) - f(x) = 3 \). If \( f(2) = 10 \), what is the value of \( f(5) \)?

To determine the value of \( f(5) \), we can use the given property of the function \( f(x + 1) - f(x) = 3 \) to find the value of \( f(3) \), \( f(4) \), and then \( f(5) \). Since \( f(2) = 10 \), applying the property:

  • \( f(3) = f(2 + 1) = f(2) + 3 = 10 + 3 = 13 \)
  • \( f(4) = f(3 + 1) = f(3) + 3 = 13 + 3 = 16 \)
  • \( f(5) = f(4 + 1) = f(4) + 3 = 16 + 3 = 19 \)
Therefore, the correct answer is \( f(5) = 19 \), which is option A.
Options B, C, and D are incorrect because they do not reflect the consistent application of the function's property for each integer step from \(2\) to \(5\).
